# Semiconductor ETL Project

## Overview
This project builds an automated ETL pipeline for semiconductor stock data using Twelve Data API, Python, PostgreSQL, and Airflow.

## Pipeline
1. Collect stock data from Twelve Data
2. Clean and preprocess with Pandas
3. Load into PostgreSQL
4. Schedule with Airflow
5. Send alerts with EmailOperator
6. Analyze data with Python
7. Build a regression model

## Tech Stack
- Python
- Pandas
- PostgreSQL
- Apache Airflow
- Matplotlib
- scikit-learn
